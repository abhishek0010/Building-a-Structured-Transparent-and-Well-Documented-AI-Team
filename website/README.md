# Multi-Agent Framework Website

Marketing and documentation website for the Advanced Multi-Agent AI Framework.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Pages

- `/` - Home page with hero, features, and quick start
- `/team` - Meet the 13 specialized AI agents
- `/tools` - MCP ecosystem tools
- `/templates` - IDE-specific configurations
- `/get-started` - Quick start guide
- `/docs` - Documentation

## Deployment

The site deploys automatically to GitHub Pages when changes are pushed to `main` in the `website/` directory.

**URL:** https://mnehmos.github.io/mnehmos.multi-agent.framework

## Tech Stack

- [Astro](https://astro.build) - Static site generator
- [Tailwind CSS](https://tailwindcss.com) - Styling
- GitHub Pages - Hosting
